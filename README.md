# claimsman

A Python project for claims management.

## Overview

Claimsman is a claims management system designed to streamline the handling, tracking, and processing of claims.

## Status

Early development — scope and architecture are still being defined.

## Requirements

- Python 3.11+

## Setup

```bash
# Clone the repository
git clone git@github.com:korobool/claimsman.git
cd claimsman

# Create a virtual environment
python -m venv .venv
source .venv/bin/activate

# Install dependencies (once defined)
pip install -r requirements.txt
```
